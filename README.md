# 🚀 프론트엔드 프로젝트

## 📋 시작하기 전에
이 프로젝트를 실행하기 위해서는 Node.js가 설치되어 있어야 합니다.

## 🛠 설치 및 실행 방법

### 1️⃣ 프로젝트 폴더로 이동
```bash
cd front
```
### 2️⃣ 필요한 패키지 설치
```bash
npm install
```
### 3️⃣ 개발 서버 실행
bash
npm start

서버가 실행되면 자동으로 http://localhost:3000 에서 확인하실 수 있습니다.

## 🌟 주요 기능
- 💻 React 기반의 모던 웹 애플리케이션
- 🎨 반응형 디자인
- ⚡ 빠른 개발 서버
